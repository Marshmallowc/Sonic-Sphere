1. UI 框架: Ant Design 或 Material-UI
2. 状态管理: Redux Toolkit
3. 路由管理: React Router

## 5. 开发规范
1. 组件命名: PascalCase (如 ProductCard)
2. 文件命名: 与组件名一致
3. CSS 类名: kebab-case (如 product-card)
4. 代码注释: 关键功能必须注释
5. Git commit: 规范commit信息
